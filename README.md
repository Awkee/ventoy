# ventoy
>一个简单的Ventoy 的启动菜单主题.


## 说明

- ventoy: Ventoy引导界面主题原始文件内容。
- release: ventoy目录的自动打包文件，打包脚本 build.sh

## 如何使用?

直接下载`release`目录下`.tar.gz`文件使用即可。


## 自定义界面参考文档

请阅读[ventoy官网关于主题自定义说明文档](https://www.ventoy.net/cn/plugin_theme.html)
